    Vuepress博客美化

    玉桂喵千寻 6/26/2022 博客美化

    # 序

    做博客经常会使用像WordPress、typecho等程序，甚至也有人会使用像 MX-space (opens new window) 个人空间这样的程序，但是对于我这种服务器渣渣的同学来说，它们非常复杂，而且也不会太稳定。但由于我之前使用过 Vuepress 文档系统，我认为我可以使用 Vuepress 进行优化实现博客的目的。为了丰富站点的功能，也可以直接使用一些现有的插件。

    # 模版

    对于已经部署好 Github 库的人来说，它非常简单。对于使用服务器的人来说，它也不会有多么复杂。

    我这里使用的是 vuepress-theme-reco (opens new window)

    # 安装方法

    它安装很简单，只需要简单的执行命令即可。

    # 安装 Vuepress

    首先安装 Vuepress 文档系统，执行：

    mkdir vuepress-starter && cd vuepress-starter && npm init && npm install -D vuepress && mkdir docs && echo '# Hello World' > blogs/README.md
    1

    按照提示进行安装即可成功安装最基础的文档系统

    再在 package.json 文件中添加一些 scripts

    {
  "scripts": {
    "dev": "vuepress dev blogs",
    "build": "vuepress build blogs"
  }
}
    1
    2
    3
    4
    5
    6

    再执行：

    npm run dev
    1

    等成功执行后访问给出的地址，显示 Hello World 就是 Vuepress 安装成功。

    # 安装模版

    执行安装模版命令：

    npm install vuepress-theme-reco
    1

    按照好后打开 blogs 文件夹，引用模版配置文件，配置站点标题等信息：

    // .vuepress/config.js

module.exports = {
  title: '你的博客名称',
  description: '你的博客的简介'
  theme: 'reco' // 博客模版，'reco' 是推荐模版，也可改成别的
}
    1

    然后修改blogs文件夹里面的 README.md 文件为：

    # 这是你的主页

---
home: true
bgImage: '/bg.png'
bgImageStyle: {
  height: '200px'
}
---

    更多首页配置请查看 模版文档-首页配置 (opens new window)

    即可将模版安装成功

    执行 npm run build 就可以将静态页面部署到 docs/.vuepress/dist 目录中

    # 2. 代码复制

    插件地址 (opens new window)

    # 安装

    npm install vuepress-plugin-code-copy
    1

    # 使用

    plugins: [['vuepress-plugin-code-copy', true]]
    1

    它会显示一个复制图标，也是最常用的代码复制插件。

    # 3. 添加著作权信息

    使用 vuepress-plugin-copyright可以禁用文字复制或者在复制时添加著作权信息。

    插件地址 (opens new window)

    # 安装

    npm install vuepress-plugin-copyright
    1

    # 使用

    plugins: [
  [
    'copyright',
    {
      noCopy: true, // 选中的文字将无法被复制
      authorName: '博客名称', // 复制后将出现著作权信息
      minLength: 40, // 如果长度超过  40 个字符
    },
  ]
]

    # 总结

    实际上使用 Vuepress 搭建博客也并不复杂，也可以把它放到 Github 中，更简单，也不需要手动执行代码实现更新文章。

    也可以放在服务器中，设置一个每三小时执行一次的计划任务，这样也可以实现每三小时自动刷新的作用

    插件繁多，可以扩展博客的功能
